# 准备工具
- Hugo
	下载后配置系统环境变量，以便方便实用Hugo。
- Git

# 初始化Hugo

## 初始化Site
Hugo new site [targetDir]

## 初始化Theme
通过Theme仓库找到自己喜欢的主题。
Clone到本地站点themes目录。
按照Theme说明完成配置。

# 迁移Jekyll博客

迁移工具失败，将Formatter全部删除了。

Theme支持原有Jekyll的Formatter，直接复制过去。
